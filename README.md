# Calculadora de Índice de massa corporal (IMC)
### Este projeto contém uma aplicação de uma calculadora de índice de massa corporal (IMC) desenvolvida em HTML, CSS e JavaScript.

### Funcionalidades:
- Calcular seu imc
- Indica seu imc
- Mostra seu nivel de peso
